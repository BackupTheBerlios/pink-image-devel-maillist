Hi All,

I have tried to build separate applications on Windows 7 using the batch file makemgw.bat

There were several problems:

1. mgw.make seems incomplete. I had to add:

  IDIR = $(PINK)/include
  CDIR = $(PINK)/src/com
  LDIR = $(PINK)/src/lib
  LIARDIR = $(PINK)/src/liar
  DDIR = $(PINK)/src/bdd
  TDIR = $(PINK)/src/tables

2. An error appears concerning chrono. I moved in file mcchrono.h the line 
	typedef struct timeval chrono;
   into the #ifndef UNIXIO clause after  #  include <time.h>

3. In cropondisk and shrinkondisk there were missing definitions (something with 64). I commented both out in the makefile.

I don't know if I changed very server things, however the operators I use work now!

I am using the pink programs in gmic (http://gmic.sourceforge.net/) where a wrapper for pink operators of type "operator infile parameter1 ... outfile" is defined. E.G. "gmic image.jpg -pink asf,5" would apply asf to the image (in fact to each channel if RGB), provided asf is in the %PATH%
