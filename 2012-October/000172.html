To compile, install Pink on ubuntu 11.10, You need the following steps:

# install the necessary dependencies
sudo apt-get install cmake-gui emacs23 libx11 xserver-xorg-dev libncurses5-dev 
libgsl0-dev libeigen2-dev g++ libboost1.46-all-dev python-dev libgsl0ldbl 
libncurses5 libboost-dev liblapack3gf libtiff4 python python-imaging python-
imaging-tk python-tk python-numpy python-matplotlib ipython python-vtk 
liblapack-dev build-essential libtiff4-dev

#then compile Pink with (you need to configure three times, this is a cmake 
limitation)
cmake -C pink_ubuntu.txt /path/to/pink
cmake -C pink_ubuntu.txt /path/to/pink
cmake -C pink_ubuntu.txt /path/to/pink

make

# after the compilation launch 
./pink.sh
# to get a Pink promt.

note: the configuration file I have used for ubuntu 11.10 is attached to this 
email and looks like follows:

# Pink's Ubuntu Config File

-msse2" CACHE STRING  "desc." )
