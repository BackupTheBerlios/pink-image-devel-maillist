Dear Pink Users, 

today I have started to work on Pink-Numpy integration. The plan is that I 
will eliminate the current cpp ujimage class and replace it with the Numpy's 
PyArrayObject class. This means a complete and transparent Python integration 
and, hopefully access to all the linalg operators provided by Numpy. The 
progress of the development can be followed at 

https://www.pinkhq.com/redmine/versions/show/6
