Hi, 
I am interested to use Pink programs also under Windows (pink-0.9.r579-Source). Perhaps there are others too interested in this non-Python installation. 

I use the Pink progams with a simple wrapper under G'MIC  (http://gmic.sourceforge.net/) either in the Terminal or under the gmic_gimp Gimp plugin.

Using makemgw.bat there were several errors. 

First some variables seemingly were not defined:
  IDIR = $(PINK)/include
  CDIR = $(PINK)/src/com
  LDIR = $(PINK)/src/lib
  LIARDIR = $(PINK)/src/liar
  DDIR = $(PINK)/src/bdd
  TDIR = $(PINK)/src/tables

Second, under my Windows 7 32 bit version mcchrono did not compile. Only commenting out a small part in mcchrono.h helped.

Third, some programs (crop/shrinkondisk) did not compile.

Perhaps caused by my virtual Windows system I had to resave the makefile as Windows file. By the way the linux build file makelin works under Mac OSX too.
