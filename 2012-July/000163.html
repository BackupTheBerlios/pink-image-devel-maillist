Named pipes do indeed exist under windows

http://en.wikipedia.org/wiki/Named_pipe
http://www.drdobbs.com/windows/using-named-pipes-to-connect-a-gui-to-a/231903148
http://www.cplusplus.com/forum/articles/11168

However we will have to write a dedicated code path for Windows. 

The named pipe is used to communicate the socket port number between the client (pink) and the server (imview), which varies from instance to instance.

I don't think I'll have time to look into this until end of August.
