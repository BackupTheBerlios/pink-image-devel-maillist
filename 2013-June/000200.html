Dear Sébastien, 

we have choosen Python as a front-end of pink, because it is easy to program 
and it is interfaced with many libraries (numpy, tk, itk, etc.) which can be 
useful to solve image processing problems. If You are unfamiliar with 
programming I would definitely suggest to use the Python tools. 

The libraries which could extend pink well depend on Your particular problem. 
If You just want to start image processing, the tutorials would suite You 
well.

Cordially, 

Laszlo
