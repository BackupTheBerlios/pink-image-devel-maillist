You can install Pink in Your Python repository. On Windows(tm) it is somewhere 
c:\Program Files\Python 2.7\x86_64\Lib\site-packages\

You have to copy libcpp_pink.pyd and the python\pink directory from the source 
tree. 

Then You can import pink by

import pink

in the interpreter.

Note: for the skeleton visualization You will also need VTK with Python 
bindings.
