> Another question is whether all the functions from the original Pink
> library are exposed to Python or not? For instance I could not find any
> function to save an image to file...

Not all the functions are supported, but saving images is definitely 
supported. There is a method called "writeimage" attached to every image 
object like:

import pink

I=pink.cpp.readimage("filename.pgm")
I.writeimage("new_filename.pgm")

There are also functions to convert pink objects from and to numpy, which can 
also be saved form Python.
